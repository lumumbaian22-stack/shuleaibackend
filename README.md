# ShuleAI Backend System v2.0

Complete school intelligence system with multi-curriculum support, teacher onboarding with school ID verification, admin approval workflow, and automated duty management.

## 🚀 Features

### Core Features
- ✅ Multi-role authentication (Super Admin, Admin, Teacher, Parent, Student)
- ✅ Multi-curriculum support (8-4-4, CBC, British, American)
- ✅ Real-time chat with WebSocket
- ✅ Academic record management with CSV upload
- ✅ Attendance tracking with historical data
- ✅ Fee and payment processing
- ✅ Alert and notification system
- ✅ School name change request workflow

### New in v2.0
- ✅ **Teacher Signup with School ID** - Teachers can sign up using school ID
- ✅ **QR Code School Access** - Scan QR code for instant school verification
- ✅ **Admin Approval Workflow** - Approve/reject teacher applications
- ✅ **Automated Duty Management** - Fair duty distribution algorithm
- ✅ **Real-time Duty Dashboard** - View duty schedules for all stakeholders
- ✅ **Check-in/Check-out System** - Track duty completion
- ✅ **Public Duty View** - Students/parents can view duty schedules
- ✅ **SMS/Email Reminders** - Automatic duty reminders
- ✅ **Teacher Duty Preferences** - Set availability and blackout dates
- ✅ **Performance Analytics** - Track duty completion rates

## 📋 Prerequisites

- Node.js >= 18.0.0
- MongoDB >= 5.0
- npm >= 9.0.0
